# 📘 Math Practice

A growing collection of browser-based mini-programs that help students practice key middle- and high-school math skills. Each activity is a standalone HTML file — just open it in a browser and start learning.

---

## 🎯 Purpose

**Math Practice** is designed for classrooms, tutoring, and independent study.  
The activities focus on procedural fluency and conceptual understanding in topics such as:

- Pre-Algebra
- Algebra
- Geometry
- Trigonometry
- Fractions, decimals, and percentages

More topics and versions will be added over time, turning this repo into a toolbox of quick, targeted math practice.

---

---

## 🚀 How to Use

1. **Download or clone** this repository.
2. Navigate to the topic you want (e.g., Algebra, Geometry).
3. **Double-click** the corresponding `.html` file to open it in your default browser.
4. Have students work through the questions, examples, or interactive elements.
5. Add or modify problems by editing the HTML file in any code editor.

No build step, frameworks, or external dependencies required.

---

## 🧑‍🏫 Classroom Ideas

- Use a specific version as a **warm-up** or **exit ticket**.
- Assign different versions to differentiate by level.
- Project an activity on the board and solve as a class.
- Let students work in pairs on laptops/Chromebooks for quick practice.

As more activities are added, teachers can build short, custom practice sets by combining multiple files.
