Functions & Graphs

What is a Function?

A function f maps every input x to exactly one output f(x). Think of it as a machine: one input in, one output out — no input can produce two different outputs.

Domain = all valid inputs x.   Range = all possible outputs f(x).

f(x) = x² Domain: all real numbers (-∞, ∞)  |  Range: [0, ∞)
g(x) = √x   Domain: [0, ∞) — can't take √ of a negative
h(x) = 1/x   Domain: all reals except x = 0

Transformations

Starting from a base function f(x), you can shift, flip, and stretch it:

Vertical Shift
f(x) + k
k > 0 → shift up   k < 0 → shift down
Horizontal Shift
f(x − h)
h > 0 → shift right   h < 0 → shift left
Reflection
−f(x)
Flips graph over the x-axis
Vertical Stretch
a · f(x)
|a| > 1 → stretch   0 < |a| < 1 → compress
Inverse Function
f⁻¹(x)
Swap x and y, then solve. Reflects over y = x
Even vs Odd
f(−x) = f(x) or −f(x)
Even: symmetric about y-axis   Odd: rotational

Function Composition

(f ∘ g)(x) = f(g(x)) — plug g(x) into f. You apply g first, then f to the result.

Important: (f ∘ g)(x) ≠ (g ∘ f)(x) in general. Order matters!

Given f(x) = x² + 1 and g(x) = 2x − 3:

1

Find (f ∘ g)(x)

f(g(x)) = f(2x−3) = (2x−3)² + 1
= 4x² − 12x + 9 + 1 = 4x² − 12x + 10
2

Find (g ∘ f)(x)

g(f(x)) = g(x²+1) = 2(x²+1) − 3
= 2x² + 2 − 3 = 2x² − 1

Evaluating Numerically

Using the same f and g: (f ∘ g)(4) — first compute g(4) = 2(4)−3 = 5, then f(5) = 5²+1 = 26.

The Unit Circle

Limits

What is a Limit?

lim(x→a) f(x) = L means: as x gets closer and closer to a (but doesn't have to equal a), the value of f(x) gets closer and closer to L.

The key insight: what f(a) actually equals doesn't matter — only what it approaches.

1

Direct Substitution — try plugging in x = a first

lim(x→3) (x² + 2x) = 3² + 2(3) = 9 + 6 = 15
2

0/0 Form — Factor & Cancel

lim(x→2) (x²−4)/(x−2) = lim (x+2)(x−2)/(x−2) = lim (x+2) = 4
3

Limits at Infinity — highest degree wins

lim(x→∞) (3x² + x)/(2x²) = lim 3/2 + 1/(2x) = 3/2

One-Sided Limits

The limit exists only if both one-sided limits agree:
lim(x→a⁻) f(x) = lim(x→a⁺) f(x) = L

If they differ, the limit Does Not Exist (DNE).

The Most Important Trig Limit

lim(x→0) sin x / x = 1 This result is needed for all derivative rules involving trig functions

Limit Laws

Assuming lim(x→a) f(x) = L and lim(x→a) g(x) = M:

LawFormulaIn Words
Sumlim [f + g] = L + MSplit the limit across addition
Differencelim [f − g] = L − MSplit the limit across subtraction
Constantlim [c · f] = c · LPull constants out front
Productlim [f · g] = L · MLimits distribute over multiplication
Quotientlim [f / g] = L / M, M≠0Limits distribute over division
Powerlim [f]ⁿ = LⁿRaise the limit to the power
Squeezeg ≤ f ≤ h → lim f = LTrap f between two functions

L'Hôpital's Rule (0/0 or ∞/∞)

If lim f/g gives 0/0 or ∞/∞, differentiate top and bottom separately:

lim(x→a) f(x)/g(x) = lim(x→a) f'(x)/g'(x)

Derivatives

The Derivative — Definition

The derivative f'(x) gives the instantaneous rate of change (slope of the tangent line) at any point x. It is defined as:

f'(x) = lim(h→0) [ f(x+h) − f(x) ] / h
Power Rule
d/dx [xⁿ] = nxⁿ⁻¹
d/dx [x⁵] = 5x⁴
Constant Rule
d/dx [c] = 0
d/dx [7] = 0
Sum / Diff Rule
d/dx[f±g] = f'±g'
Differentiate term by term
Product Rule
d/dx[fg] = f'g + fg'
"First · d(second) + second · d(first)"
Quotient Rule
d/dx[f/g] = (f'g−fg')/g²
"Low d(high) − high d(low), over low²"
Trig Derivatives
d/dx[sin x] = cos x
d/dx[cos x] = −sin x
d/dx[tan x] = sec²x
Exponential
d/dx[eˣ] = eˣ
d/dx[aˣ] = aˣ · ln a
Logarithm
d/dx[ln x] = 1/x
d/dx[logₐx] = 1/(x ln a)

Chain Rule

For a composite function y = f(g(x)), differentiate the outer function evaluated at the inner, times the derivative of the inner:

d/dx [ f(g(x)) ] = f'(g(x)) · g'(x)

Example: Differentiate y = (3x² + 1)⁵

1

Identify outer = ( )⁵ and inner = 3x² + 1

2

Derivative of outer: 5( )⁴ — keep inner inside

5(3x²+1)⁴
3

Multiply by derivative of inner: d/dx[3x²+1] = 6x

dy/dx = 5(3x²+1)⁴ · 6x = 30x(3x²+1)⁴

Chain Rule + Trig

d/dx [ sin(u) ] = cos(u) · u'

Example: d/dx [sin(x³)] = cos(x³) · 3x² = 3x²cos(x³)

Derivative Applications

Critical Points & Extrema

A critical point occurs where f'(x) = 0 or f'(x) is undefined. These are candidates for local maxima, minima, or saddle points.

First Derivative Test: if f' changes from + to − → local max. From − to + → local min.

Second Derivative Test: if f''(c) < 0 → local max. f''(c) > 0 → local min. f''(c) = 0 → inconclusive.

Example: Find the local extrema of f(x) = x³ − 3x² − 9x + 5

1

Find f'(x) and set equal to zero

f'(x) = 3x² − 6x − 9 = 3(x² − 2x − 3) = 3(x−3)(x+1)
Critical points: x = 3 and x = −1
2

Second derivative test: f''(x) = 6x − 6

f''(3) = 12 > 0 → Local minimum at x = 3
f''(−1) = −12 < 0 → Local maximum at x = −1

Concavity & Inflection Points

f''(x) > 0 → concave up (cup shape). f''(x) < 0 → concave down (hill shape). An inflection point is where concavity changes (f'' changes sign).

Mean Value Theorem

If f is continuous on [a,b] and differentiable on (a,b), then there exists at least one c in (a,b) such that:

f'(c) = [ f(b) − f(a) ] / ( b − a )

Integrals

The Antiderivative (Indefinite Integral)

Integration is the reverse of differentiation. The indefinite integral of f(x) gives all functions whose derivative is f(x). The constant C represents any constant (which disappears when you differentiate).

∫ f(x) dx = F(x) + C   where   F'(x) = f(x)
Power Rule
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C
∫ x³ dx = x⁴/4 + C
Constant
∫ k dx = kx + C
∫ 5 dx = 5x + C
Sum/Diff
∫[f±g] dx = ∫f dx ± ∫g dx
Integrate term by term
Trig Integrals
∫ sin x dx = −cos x + C
∫ cos x dx = sin x + C
Exponential
∫ eˣ dx = eˣ + C
∫ e²ˣ dx = ½e²ˣ + C
Reciprocal
∫ 1/x dx = ln|x| + C
Note the absolute value bars

Fundamental Theorem of Calculus — Part 1

If F(x) = ∫ₐˣ f(t) dt, then F'(x) = f(x). In other words, differentiating a definite integral with a variable upper bound just gives back the integrand.

Fundamental Theorem of Calculus — Part 2

If F is any antiderivative of f, then the definite integral from a to b equals:

∫ₐᵇ f(x) dx = F(b) − F(a)

Example: Evaluate ∫₁³ (3x² + 2) dx

1

Find the antiderivative

F(x) = x³ + 2x (+ C, but constants cancel)
2

Evaluate F(3) − F(1)

F(3) = 27 + 6 = 33
F(1) = 1 + 2 = 3
∫₁³ (3x²+2) dx = 33 − 3 = 30
